# پروژه خلاصه‌ساز صورتحساب

این پروژه یک ابزار تحت وب برای پردازش و خلاصه‌سازی داده‌های صورتحساب است.

## ویژگی‌ها
- ورودی داده متنی (کپی از اکسل)
- تشخیص خودکار ستون‌ها
- تبدیل هوشمند اعداد فارسی و حذف کاراکترهای غیر عددی
- گروه‌بندی بر اساس ستون «کد تنوع»
- نمایش جمع کل ستون‌های عددی

## قوانین توسعه
- زبان پیش‌فرض وب‌سایت فارسی است.
- همیشه ساختار RTL حفظ شود.
- در صورت تغییر منطق پردازش، فایل `utils/dataHelpers.js` بررسی شود.
